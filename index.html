What is Outbound? Outbound 是什么游戏？

Outbound is a cozy open-world exploration and survival-crafting game set in a utopian near future. You start with an empty electric camper van and gradually turn it into a mobile home.《Outbound》是一款设定在未来乌托邦背景的舒适开放世界探索与生存建造游戏。玩家从一辆空荡荡的电动房车开始，逐步把它改造和移动之家。
