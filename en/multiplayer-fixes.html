Connection Issues连接问题

Can't connect to friends? Try these fixes无法连接好友？尝试以下修复

❌ Cannot Join Friend's Game无法加入好友游戏

Try First:首先尝试：

  • Check both players are on same Steam region检查双方玩家在同一Steam区域
  • Verify host has "Joinable" status enabled验证主机已开启"可加入"状态
  • Restart game and Steam completely完全重启游戏和Steam

If Still Not Working:如果仍然无效：

  • Check firewall/antivirus exceptions检查防火墙/杀毒软件例外
  • Port forward: UDP 27036, 27031端口转发：UDP 27036, 27031
  • Try disabling VPN尝试禁用VPN

⚠️ Lagging or Disconnecting卡顿或断开连接

Network Optimization:网络优化：

  • Host should have stable upload speed主机应有稳定的上传速度
  • Close bandwidth-heavy applications关闭占用带宽的应用程序
  • Use wired connection over WiFi when possible尽可能使用有线连接而非WiFi

Recommended host specs for 4-player:4人游戏推荐主机规格： 10+ Mbps upload

🌐 NAT Type IssuesNAT类型问题

NAT type affects peer-to-peer connection quality. Strict NAT often causes connection failures.NAT类型影响点对点连接质量。严格NAT通常导致连接失败。

NAT Types (Best to Worst):NAT类型（从好到差）：

Open (Best) Moderate Strict (Worst)

To improve NAT: enable UPnP on router, port forward, or use VPN with port forwarding.改善NAT：开启路由器UPnP、端口转发、或使用带端口转发的VPN。

🔀 Steam Family Sharing ConflictSteam家庭共享冲突

Family sharing can cause connection issues between accounts.家庭共享可能导致账户之间的连接问题。

Fix:修复：

  1. Host must own their own copy (not using shared)主机必须拥有自己的副本（不使用共享）
  2. Each player should own or legitimately share每个玩家应拥有或合法共享
  3. Avoid same device shared sessions避免同一设备共享会话

Host Issues主机问题

Problems when you're the host当你作为主机时的问题

🚪 Cannot Set Game to Joinable无法将游戏设为可加入

The "Make Joinable" option sometimes doesn't appear or won't work."设为可加入"选项有时不出现或无法工作。

Workarounds:解决方案：

  1. Restart Outbound completely完全重启Outbound
  2. Check you're not in a loading screen检查你不在加载画面中
  3. Try inviting via Steam overlay (Shift+Tab)尝试通过Steam覆盖层邀请（Shift+Tab）
  4. Exit to main menu, then re-make joinable退出到主菜单，然后重新设为可加入

🔄 Host Migration Failed主机迁移失败

When host leaves, game should migrate to next player. Sometimes this fails.当主机离开时，游戏应迁移到下一个玩家。有时这会失败。

If Host Migration Fails:如果主机迁移失败：

  • All progress since last save is lost上次存档后的所有进度将丢失
  • Save frequently to minimize loss频繁存档以最小化损失
  • Host should save before leaving主机离开前应存档

This is a known issue - devs are working on fix.这是已知问题 - 开发者在修复中。

👥 Player Dropping Out玩家掉线

Players getting kicked unexpectedly during gameplay.游戏过程中玩家意外掉线。

Prevention:预防：

  • Host: Don't alt-tab or minimize game主机：不要切换或最小化游戏
  • All: Stable internet, close other downloads所有人：稳定网络，关闭其他下载
  • All: Update to latest game version所有人：更新到最新游戏版本

🔒 Password Protected Games密码保护的游戏

Setting a password for private sessions.为私人会话设置密码。

Go to host menu → "Set Password" → enter 4-digit code. Share with friends before they try to join.进入主机菜单 → "设置密码" → 输入4位数代码。在朋友尝试加入前分享给他们。

Host-Only Achievements仅主机成就

Known issue: achievements not unlocking for non-host players已知问题：非主机玩家的成就无法解锁

⚠️

Known Bug Report已知bug报告

Multiple players report that host-only achievements are not unlocking for joiners in co-op mode. This appears to be a game bug, not a user error.多个玩家报告联机模式中非主机玩家的仅主机成就不解锁。这似乎是游戏bug，不是用户错误。

🐛 Confirmed Bug ⏳ Devs Working on Fix

🔄 Workaround: Host Rotation🔄 变通方法：主机轮换

To earn host-only achievements, each player must host a session.为了获得仅主机成就，每个玩家必须主持一个会话。

Steps:步骤：

  1. Player A hosts, everyone plays玩家A主持，大家玩
  2. Player A saves game (checkpoint)玩家A存档（检查点）
  3. Player B hosts new session from save玩家B从存档主持新会话
  4. Repeat for each player对每个玩家重复

📝 Report to Developers📝 向开发者报告

Help fix this faster by reporting with details.通过详细报告帮助更快修复。

Report Channels:报告渠道：

  • Discord #bug-reports channel
  • Steam Community Forums
  • r/Outbound

Include: game version, platform, steps to reproduce, screenshot if possible.包括：游戏版本、平台、重现步骤、可能的截图。

Common Bugs常见bug

Fixes for frequently reported issues修复频繁报告的问题

🔥 Common

Game Freezes on Load Screen加载画面时游戏冻结

Quick Fixes:快速修复：

  • Verify game files in Steam在Steam中验证游戏文件
  • Update GPU drivers更新GPU驱动
  • Run as Administrator以管理员身份运行

If Persists:如果持续：

  • Delete local save files, restart删除本地存档文件，重启
  • Check for pending updates检查是否有待处理更新
⚠️ Moderate

Items Not Syncing Between Players物品在玩家之间不同步

Picked up items appearing/disappearing for different players.捡起的物品对不同玩家出现/消失。

Workaround:变通方法：

  • Host: Check that all players have same items主机：检查所有玩家是否有相同物品
  • Host: Redistribute synced items manually主机：手动重新分配同步的物品
  • Known issue - report to devs if frequent已知问题 - 如果频繁发生则向开发者报告
🔧 Fixable

Cant Craft After Joining Mid-Game中途加入后无法合成

New joiners sometimes can't access workbench or crafting.新加入者有时无法访问工作台或合成。

Fix:修复：

  1. Host: Walk to workbench with joiner主机：与加入者一起走到工作台
  2. Joiner: Stand within interaction range加入者：站在交互范围内
  3. Host: Show interact prompt to joiner主机：向加入者显示交互提示
  4. Both: Wait for sync (5-10 seconds)双方：等待同步（5-10秒）

Matchmaking Issues匹配问题

Finding and joining random players查找和加入随机玩家

Public Matchmaking Not Working公开匹配不工作

Outbound uses peer-to-peer, not dedicated servers. Public matchmaking finds players on same network type.Outbound使用点对点而非专用服务器。公开匹配查找相同网络类型的玩家。

  • Better NAT = more matches更好的NAT = 更多匹配
  • Off-peak hours = fewer matches非高峰时段 = 更少匹配
  • Join Discord LFG for easier co-op更易找到联机

Private Sessions私人会话

For playing with friends only, use private session.仅与朋友游玩时，使用私人会话。

  1. Host: Main Menu → Play → Private Session主机：主菜单 → 开始游戏 → 私人会话
  2. Host: Set password (optional but recommended)主机：设置密码（可选但推荐）
  3. Friends: Steam overlay invite or direct join朋友：Steam覆盖层邀请或直接加入
