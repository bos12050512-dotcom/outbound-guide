Is Outbound on Xbox Game Pass?

Yes, Outbound is available on Xbox Game Pass for both PC (via Xbox app / Microsoft Store) and Xbox consoles. The game was added to the Game Pass catalog around its official 1.0 launch in mid-2026, making it accessible to millions of subscribers at no additional cost beyond the subscription fee.

✅ Subscription Tiers That Include Outbound:

  • • Xbox Game Pass for PC
  • • Xbox Game Pass Ultimate (Console + PC + Cloud)
  • • Xbox Game Pass Console (Xbox Series X|S and Xbox One)

If you are an active Game Pass subscriber, you can install Outbound directly from the Xbox app on Windows or from the Microsoft Store / Xbox Store on console. The file size is approximately 4-6 GB, so ensure you have enough storage space available.

How to Play Outbound on Xbox Game Pass

Getting started with Outbound on XGP is straightforward. Follow these steps depending on your platform:

On PC (Windows 10/11)

  1. Open the Xbox app on your PC (download from Microsoft Store if not installed).
  2. Sign in with your Microsoft account that has an active Game Pass subscription.
  3. Search for "Outbound" in the search bar.
  4. Click Install to download the game.
  5. Launch from the Xbox app or from your Start Menu once installation completes.

On Xbox Console (Series X|S / One)

  1. Press the Xbox button and open the Microsoft Store or Game Pass tab.
  2. Search for "Outbound" and select the Game Pass version.
  3. Click Install to begin downloading.
  4. Once installed, launch from My Games & Apps.

💡 Pro Tip

Enable "Auto-Update" in your Xbox app settings so Outbound stays current with the latest patches without manual intervention.

Cross-Play & Cross-Save

One of the most common questions from the Steam Community is whether Xbox Game Pass players can play with Steam players. Here is what you need to know about cross-platform functionality in Outbound.

Cross-Play (Multiplayer Between Platforms)

Outbound supports cross-play between Xbox Game Pass (PC) and Steam PC players. This means if you are playing through the Xbox app on PC, you can join multiplayer sessions hosted by friends on Steam, and vice versa.

How cross-play works: The game uses an in-game invite code system. The host generates a session code from the pause menu, and other players enter this code to join — regardless of whether they are on Steam or Xbox Game Pass PC.

However, console-to-PC cross-play is currently limited. Xbox console players (Series X|S / One) can play with other Xbox console players and Xbox Game Pass PC players through the Xbox network, but direct cross-play between Xbox console and Steam PC via invite codes may not be fully supported at this time. Always check the latest patch notes for updates on this feature.

Cross-Save (Save File Transfer)

Outbound supports cross-save between Xbox Game Pass PC and Xbox console through Xbox cloud saves. Your progress is automatically synced to your Microsoft account, allowing you to continue your journey on either platform.

⚠️ Important Limitation

Cross-save between Xbox Game Pass and Steam is not supported. Save files are stored in different formats and locations (Microsoft cloud vs. local Steam userdata). If you switch from XGP to Steam (or buy the game on Steam after trying it on Game Pass), you will need to start a new save file.

Multiplayer on Xbox Game Pass

Multiplayer is one of Outbound's standout features, supporting up to 4 players co-op. Xbox Game Pass players have reported some unique challenges when setting up multiplayer sessions, particularly when mixing platforms. This section covers the most common scenarios and solutions based on community discussions.

How to Host a Multiplayer Session

  1. Launch Outbound and load your save file.
  2. Open the pause menu and select Multiplayer.
  3. Click Host Session to create a lobby.
  4. Share the generated invite code with your friends (4-digit or alphanumeric code).
  5. Your friends enter the code in their Multiplayer menu under Join Session.

Common XGP Multiplayer Issues (From Steam Discussions)

❌ "Friends can't join my XGP session"

This is the most reported issue. If you are on XGP PC and your friend is on Steam, ensure both of you are on the exact same game version. The Xbox app and Steam may receive patches at slightly different times. Check the version number in the main menu bottom-right corner.

❌ "Connection timeout / session not found"

Xbox Game Pass PC uses a different networking layer than Steam. Try having the Steam player host instead of the XGP player — many users report better success with Steam-hosted sessions. Also ensure Windows Firewall is not blocking the Xbox app.

❌ "NAT type strict on Xbox console"

Console players with Strict or Moderate NAT types may have trouble connecting to PC-hosted sessions. Go to Xbox Settings > Network > Test NAT Type. If it shows Strict, try enabling UPnP on your router or setting a port forward for Xbox Live.

Best Practices for Stable Co-op

  • Use a wired connection if possible — Wi-Fi can cause desync in larger camps.
  • Designate one permanent host to avoid world-state conflicts.
  • Save frequently by sleeping in your van bed — progress only saves for the host.
  • Avoid simultaneous building — have one player place structures while others gather.
  • Restart the game if you experience inventory desync or invisible items.

Known Issues & Fixes

Based on community reports from both the Xbox app reviews and Steam Community Discussions, here are the most common issues affecting Xbox Game Pass players and their workarounds.

Game won't launch from Xbox app

Some players report the game crashing immediately or showing a black screen.

Fix:

  • Update the Xbox app and Gaming Services through Microsoft Store.
  • Reset the game: Windows Settings > Apps > Outbound > Advanced options > Reset.
  • Ensure Windows is fully updated (especially the Gaming Services component).
  • Run the Xbox app as Administrator.

Controller not recognized (XGP PC)

Xbox controllers may not be detected in-game even though Windows sees them.

Fix:

  • Disconnect and reconnect the controller after launching the game.
  • Use a wired USB connection instead of Bluetooth.
  • Disable Steam Input if you also have Steam running — it can conflict.
  • Update controller firmware via Xbox Accessories app.

Progress not syncing between PC and console

Cloud save sync can occasionally fail between Xbox Game Pass PC and Xbox console.

Fix:

  • Ensure you are signed into the same Microsoft account on both devices.
  • On Xbox console: My Games & Apps > Outbound > Menu > Manage game > Saved data > Sync now.
  • Close the game fully on one platform before opening on the other.
  • Allow 5-10 minutes for cloud sync after closing the game.

Missing DLC / Deluxe Edition content

Xbox Game Pass includes the base game only. Deluxe Edition cosmetics or DLC packs are not included.

Fix:

  • Deluxe Edition items can be purchased separately from the Microsoft Store.
  • Game Pass subscribers usually receive a 10-20% discount on DLC purchases.
