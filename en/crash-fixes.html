Game Won't Launch / Crashes on Startup

The most reported issue on Steam forums

Why this happens

Outbound uses the Unity engine with IL2CPP scripting backend. Common causes include corrupted game files, outdated VC++ runtimes, missing DirectX components, and antivirus interference. The issue spiked after the July 2026 patch.

1 Verify Game Files

Corrupted or missing files are the #1 cause of startup crashes.

  1. Open Steam → Library
  2. Right-click Outbound → Properties
  3. Go to Installed Files tab
  4. Click Verify integrity of game files
  5. Wait for completion, then restart Steam
✅ This fixes ~60% of startup crashes

2 Update Visual C++ Redistributables

IL2CPP requires the latest VC++ runtime to function.

  1. Download VC++ Redistributable 2015-2022 (x64) from Microsoft
  2. Run the installer and follow prompts
  3. Restart your PC
  4. Launch Outbound again
💡 Download from aka.ms/vs/17/release/vc_redist.x64.exe

3 Run as Administrator

Permission issues can block the game from reading critical files.

  1. Right-click Outbound.exe or the Steam shortcut
  2. Select Properties → Compatibility
  3. Check Run this program as an administrator
  4. Click Apply → OK

4 Disable Overlay & Background Apps

Steam Overlay, Discord, and GeForce Experience can conflict with Unity.

  1. Steam: Settings → In-Game → Uncheck Enable Steam Overlay
  2. Discord: Disable In-Game Overlay
  3. Close MSI Afterburner / Rivatuner if running
  4. Temporarily disable GeForce Experience overlay

🛡️ Prevention

  • Add Outbound folder to antivirus exclusions
  • Don't alt-tab during the initial loading sequence
  • Keep GPU drivers updated (NVIDIA 566+ / AMD 24.12+)
  • Install the game on an SSD, not a mechanical hard drive

Black Screen on Launch

Game runs but display stays black

Why this happens

Usually caused by incorrect display resolution, fullscreen mode conflicts, or the game rendering on an integrated GPU instead of your dedicated graphics card. Dual-GPU laptops are especially prone to this.

1 Force Dedicated GPU

Laptops often default to integrated graphics.

NVIDIA:

NVIDIA Control Panel → Manage 3D Settings → Program Settings → Select Outbound → Preferred GPU: High-performance NVIDIA processor

AMD:

AMD Software → Graphics → Add Outbound → Set to High Performance

2 Delete Display Settings

Reset the game's display configuration.

# Delete or rename this file:

%AppData%\Outbound\settings.json

The game will recreate it with default settings on next launch.

3 Launch in Windowed Mode

Bypass fullscreen-exclusive mode conflicts.

  1. Right-click Outbound in Steam Library
  2. Properties → General
  3. In Launch Options, type: -windowed -w 1920 -h 1080
  4. Launch the game

4 Update Graphics Drivers

Black screens are often resolved with latest drivers.

NVIDIA Drivers → AMD Drivers → Intel Drivers →

🛡️ Prevention

  • Set your monitor to native resolution before launching
  • Disable HDR mode if you experience black screen after logo
  • Don't use Alt+Enter to toggle fullscreen during loading
  • For dual-monitor setups, try disconnecting the secondary monitor temporarily

Multiplayer Connection Issues

Can't join friends or getting disconnected

Why this happens

Outbound uses peer-to-peer (P2P) networking via Steam's relay servers. Connection issues are usually caused by NAT type conflicts, firewall blocking, outdated game versions between players, or unstable host internet.

1 Check NAT Type

Strict NAT prevents P2P connections.

OpenBest - no issues expected
ModerateWorks with most players
StrictCannot host; can only join Open hosts

Check in Steam: View → Settings → In-Game or use your router's admin panel.

2 Port Forwarding

Open these ports on your router for reliable hosting.

UDP: 27015-27050

TCP: 27015-27030

UDP: 3478, 4379, 4380

Or enable UPnP on your router to let Steam handle this automatically.

3 Same Game Version

Version mismatch is a common cause of failed joins.

  1. All players: Check version in main menu (bottom-left corner)
  2. If different: Restart Steam to force update
  3. Verify game files if update doesn't appear
  4. Wait for Steam to finish downloading before launching

4 Firewall & Antivirus

Windows Firewall often blocks P2P games.

  1. Open Windows Security → Firewall
  2. Click Allow an app through firewall
  3. Add Outbound.exe for both Private and Public
  4. Temporarily disable third-party AV to test

🛡️ Prevention

  • Host should have at least 5 Mbps upload speed
  • Use wired Ethernet instead of WiFi when hosting
  • Close bandwidth-heavy apps (streaming, downloads) while playing
  • Save game before inviting players to minimize loss on disconnect

Save File Corruption

Progress lost, won't load, or saves disappear

Why this happens

Corruption usually occurs from force-closing the game during auto-save, power loss, cloud sync conflicts (especially with Steam Cloud + multiple PCs), or disk errors. The July 2026 patch introduced a new save format that occasionally conflicts with old backups.

1 Restore from Backup

Outbound creates automatic backups every 30 minutes.

# Save locations:

%AppData%\Outbound\Saves\

%AppData%\Outbound\Saves_Backup\

  1. Navigate to the backup folder
  2. Find the most recent .sav file before corruption
  3. Copy it to the main Saves folder
  4. Rename to match the original save name

2 Disable Steam Cloud Temporarily

Cloud sync conflicts can overwrite valid saves.

  1. Steam → Right-click Outbound → Properties
  2. General tab → Uncheck Keep games saves in Steam Cloud
  3. Restore local save from backup
  4. Launch game to verify it loads
  5. Re-enable Steam Cloud after confirming it works

3 Check Disk for Errors

Bad sectors can corrupt save files.

  1. Open File Explorer → Right-click your game drive
  2. Properties → Tools tab
  3. Click Check under Error checking
  4. Run chkdsk /f if prompted
  5. Restart your PC to complete the scan

4 Use Save Editor (Advanced)

If the save won't load at all, a save editor may recover it.

⚠️ Warning

Always back up your saves before editing. Community save editors are available on r/Outbound and Nexus Mods, but use at your own risk.

🛡️ Prevention

  • Never Alt+F4 or force-close during the auto-save icon
  • Manually save at safe locations before quitting
  • Back up your save folder weekly to cloud storage or external drive
  • Don't play the same save on multiple PCs without waiting for cloud sync

Performance Issues & Stuttering

Low FPS, frame drops, and micro-stutters

Why this happens

Outbound is an open-world Unity game with dynamic weather, day-night cycles, and physics-driven objects. Stuttering often comes from shader compilation, loading assets from HDD, CPU bottlenecks during world generation, or background processes stealing resources.

1 Best Settings for FPS

Shadow QualityLow / Off
View DistanceMedium
Texture QualityHigh
V-SyncOff
Anti-AliasingFXAA
Motion BlurOff

2 Fix Stuttering

Persistent stutter fixes for Unity games.

  1. Set Power Plan to High Performance in Windows
  2. In NVIDIA/AMD Control Panel, set Power Management to Prefer Maximum Performance
  3. Disable Fullscreen Optimizations for Outbound.exe
  4. Cap FPS to your monitor's refresh rate using RTSS or in-game limiter
  5. Close Chrome/Edge tabs (they eat RAM)

3 Install on SSD

HDDs cause massive hitching during open-world streaming.

Recommended:

NVMe SSD with 3000+ MB/s read speed. SATA SSD is acceptable. Avoid mechanical hard drives for this game.

4 Shader Preloading

Unity shader compilation causes micro-stutter.

  1. Launch the game and let it sit at the main menu for 5 minutes
  2. Start a new game, stand still for 2-3 minutes
  3. Walk around the starting area to force shader compilation
  4. Restart the game after this "warm-up" session

Subsequent sessions will be much smoother.

🛡️ Prevention

  • Keep GPU drivers updated (check monthly)
  • Ensure at least 16GB RAM (8GB causes frequent stutter)
  • Don't run OBS or streaming software on the same PC if FPS is already low
  • Defragment HDDs monthly (not needed for SSDs)

Controller Not Working

Xbox, PlayStation, or third-party controllers unresponsive

Why this happens

Unity's input system can fail to detect controllers if Steam Input is interfering, if the controller was connected after launch, or if drivers for third-party controllers (like DS4Windows) are conflicting with native Steam Input.

1 Disable Steam Input

Steam Input often conflicts with native controller support.

  1. Steam → Right-click Outbound → Properties
  2. Controller tab
  3. Change Override for Outbound to Disable Steam Input
  4. Disconnect and reconnect your controller
  5. Launch the game
✅ Fixes most Xbox and PlayStation controller issues

2 Connect Before Launch

Unity sometimes fails to detect hot-plugged controllers.

  1. Close Outbound completely
  2. Connect your controller via USB or Bluetooth
  3. Wait 10 seconds for Windows to recognize it
  4. Verify it appears in Windows Settings → Bluetooth & devices
  5. Launch Outbound

3 DS4Windows / DualSense Users

DualShock 4 and DualSense need special handling.

  1. Close DS4Windows completely
  2. In Steam, enable PlayStation Configuration Support
  3. Keep Steam Input Enabled for PlayStation controllers
  4. Or use DS4Windows in Hide DS4 Controller mode with Steam Input disabled

Don't run both DS4Windows and Steam Input simultaneously.

4 Test in Big Picture Mode

Steam Big Picture provides better controller support.

  1. Launch Steam Big Picture Mode
  2. Navigate to Outbound in your library
  3. Launch from Big Picture
  4. Check if controller works in this mode

If it works in Big Picture but not normal mode, Steam Input configuration is the culprit.

🛡️ Prevention

  • Use wired connection for competitive play (less latency, no disconnects)
  • Update controller firmware (Xbox Accessories app / DS4Windows)
  • Don't connect multiple controllers unless playing local co-op
  • Keep Bluetooth drivers updated to avoid input lag
