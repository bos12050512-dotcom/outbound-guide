Outbound 是否在 Xbox Game Pass 上？

是的，Outbound 已登陆 Xbox Game Pass，支持 PC 端（通过 Xbox 应用 / Microsoft Store）和 Xbox 主机端。该游戏于 2026 年中正式版 1.0 发布时加入 Game Pass 游戏库，订阅用户无需额外付费即可游玩。

✅ 包含 Outbound 的订阅档位：

  • • Xbox Game Pass for PC（PC 版）
  • • Xbox Game Pass Ultimate（主机 + PC + 云游戏）
  • • Xbox Game Pass Console（Xbox Series X|S 和 Xbox One）

如果你是活跃的 Game Pass 订阅用户，可以直接通过 Windows 上的 Xbox 应用或主机上的 Microsoft Store / Xbox Store 安装 Outbound。游戏文件大小约为 4-6 GB，请确保有足够的存储空间。

如何在 Xbox Game Pass 上游玩 Outbound

在 XGP 上开始 Outbound 之旅非常简单。根据你的平台，按照以下步骤操作：

PC 端（Windows 10/11）

  1. 在电脑上打开 Xbox 应用（如未安装，可从 Microsoft Store 下载）。
  2. 使用拥有有效 Game Pass 订阅的 Microsoft 账号登录。
  3. 在搜索栏中搜索 "Outbound"。
  4. 点击 安装 按钮下载游戏。
  5. 安装完成后，从 Xbox 应用或开始菜单启动游戏。

Xbox 主机端（Series X|S / One）

  1. 按 Xbox 按钮，打开 Microsoft Store 或 Game Pass 标签页。
  2. 搜索 "Outbound" 并选择 Game Pass 版本。
  3. 点击 安装 开始下载。
  4. 安装完成后，从"我的游戏和应用"中启动。

💡 专业提示

在 Xbox 应用设置中启用"自动更新"，这样 Outbound 就能在发布新补丁时自动保持最新，无需手动操作。

跨平台联机与存档同步

Steam 社区中最常见的问题之一是：Xbox Game Pass 玩家能否与 Steam 玩家一起联机？以下是关于 Outbound 跨平台功能的详细说明。

跨平台联机（不同平台间的多人游戏）

Outbound 支持 Xbox Game Pass（PC）与 Steam PC 玩家之间的跨平台联机。这意味着如果你通过 PC 端的 Xbox 应用游玩，可以加入 Steam 好友创建的多人游戏，反之亦然。

跨平台联机机制：游戏使用内置的邀请码系统。房主从暂停菜单生成一个会话代码，其他玩家输入该代码即可加入 —— 无论对方是在 Steam 还是 Xbox Game Pass PC 上游玩。

不过，主机与 PC 之间的跨平台联机目前有一定限制。Xbox 主机玩家（Series X|S / One）可以通过 Xbox 网络与其他 Xbox 主机玩家和 Xbox Game Pass PC 玩家联机，但 Xbox 主机与 Steam PC 之间通过邀请码的直接跨平台联机可能尚未完全支持。请随时关注最新补丁说明以获取该功能的更新。

跨平台存档（存档文件转移）

Outbound 支持 Xbox Game Pass PC 与 Xbox 主机之间的跨平台存档，通过 Xbox 云存档自动同步。你的进度会自动保存到 Microsoft 账号中，让你可以在任一平台上继续冒险。

⚠️ 重要限制

Xbox Game Pass 与 Steam 之间的跨平台存档不被支持。存档文件存储在不同的格式和位置（Microsoft 云端 vs. 本地 Steam 用户数据）。如果你从 XGP 切换到 Steam（或在 Game Pass 试玩后购买 Steam 版），需要重新开始新的存档。

XGP 多人联机指南

多人联机是 Outbound 最亮眼的功能之一，支持最多 4 人合作。Xbox Game Pass 玩家在设置多人游戏时报告了一些独特的挑战，特别是在混合不同平台时。本节根据社区讨论整理了最常见的场景和解决方案。

如何创建多人游戏房间

  1. 启动 Outbound 并加载你的存档。
  2. 打开暂停菜单，选择 多人游戏。
  3. 点击 创建房间 来建立联机大厅。
  4. 将生成的 邀请码 分享给你的好友（4 位数字或字母数字组合）。
  5. 你的好友在多人游戏菜单中选择 加入房间 并输入邀请码。

XGP 常见联机问题（来自 Steam 讨论区）

❌ "好友无法加入我的 XGP 房间"

这是报告最多的问题。如果你在使用 XGP PC 版，而好友在 Steam 上，请确保你们的游戏版本完全一致。Xbox 应用和 Steam 的补丁发布时间可能略有不同。检查主菜单右下角的版本号。

❌ "连接超时 / 找不到房间"

Xbox Game Pass PC 使用的网络层与 Steam 不同。尝试让 Steam 玩家担任房主而非 XGP 玩家 —— 许多用户报告 Steam 房主的成功率更高。同时确保 Windows 防火墙没有阻止 Xbox 应用。

❌ "Xbox 主机 NAT 类型为严格"

NAT 类型为"严格"或"中等"的主机玩家可能难以连接 PC 房主创建的房间。前往 Xbox 设置 > 网络 > 测试 NAT 类型。如果显示"严格"，尝试在路由器上启用 UPnP 或为 Xbox Live 设置端口转发。

稳定联机的最佳实践

  • 尽可能使用有线网络 —— Wi-Fi 在大型营地中可能导致不同步。
  • 指定一名固定房主 —— 避免世界状态冲突。
  • 频繁保存进度 —— 在房车床上睡觉保存，只有房主的进度会被保存。
  • 避免同时建造 —— 让一名玩家负责放置建筑，其他人负责收集资源。
  • 重启游戏 —— 如果遇到物品栏不同步或隐形物品问题，尝试重启。

已知问题与修复方法

根据 Xbox 应用评论和 Steam 社区讨论区的玩家反馈，以下是最常见的 Xbox Game Pass 玩家遇到的问题及其解决方法。

游戏无法从 Xbox 应用启动

部分玩家报告游戏启动后立即崩溃或黑屏。

解决方法：

  • 通过 Microsoft Store 更新 Xbox 应用和 Gaming Services。
  • 重置游戏：Windows 设置 > 应用 > Outbound > 高级选项 > 重置。
  • 确保 Windows 系统完全更新（尤其是 Gaming Services 组件）。
  • 以管理员身份运行 Xbox 应用。

手柄无法识别（XGP PC 版）

Windows 可以识别 Xbox 手柄，但游戏内无法使用。

解决方法：

  • 启动游戏后断开并重新连接手柄。
  • 使用有线 USB 连接代替蓝牙。
  • 如果同时运行了 Steam，禁用 Steam 输入 —— 可能会产生冲突。
  • 通过 Xbox Accessories 应用更新手柄固件。

PC 与主机之间进度无法同步

Xbox Game Pass PC 与 Xbox 主机之间的云存档同步偶尔可能失败。

解决方法：

  • 确保你在两台设备上登录的是同一个 Microsoft 账号。
  • 在 Xbox 主机上：我的游戏和应用 > Outbound > 菜单 > 管理游戏 > 已保存数据 > 立即同步。
  • 在另一个平台上完全关闭游戏后再打开。
  • 关闭游戏后等待 5-10 分钟，让云存档完成同步。

缺少 DLC / 豪华版内容

Xbox Game Pass 仅包含基础游戏。豪华版 cosmetic 道具或 DLC 包不包含在内。

解决方法：

  • 豪华版物品可以单独从 Microsoft Store 购买。
  • Game Pass 订阅用户购买 DLC 通常可享受 10-20% 的折扣。
