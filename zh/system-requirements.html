Outbound PC 配置要求

来自 Steam 商店页面的官方最低/推荐配置，加上常见显卡的帧数预期。

[ 广告位 - 顶部横幅 ]

最低配置

  • 系统：Windows 10 64 位
  • CPU：Intel Core i3-10100F / AMD Ryzen 3 3100
  • 内存：4 GB
  • 显卡：NVIDIA GTX 1050 / AMD RX 570
  • DirectX：11
  • 存储：16 GB

推荐配置

  • 系统：Windows 11 64 位
  • CPU：Intel i3-12100F / Ryzen 7 1700 或更好
  • 内存：8 GB
  • 显卡：NVIDIA RTX 3060 / AMD RX 7600 XT 或更好
  • DirectX：12
  • 存储：16 GB

我的电脑能跑 Outbound 吗？

  • 核显笔记本：大概率不行。最低显卡要求是 GTX 1050。
  • 办公主机 + GTX 1050 / RX 570：可以，1080p 低画质 30–45 帧。
  • 中端游戏机 1660 / 3060：1080p 高画质流畅。
  • 4K 游戏：官方未明示。建议 4070 及以上才能稳定 4K。
