# Steam 社区推广策略

## 回复模板

### 1. 新手求助回复
**场景**: 有人在讨论区问"刚开始玩，应该先做什么？"

**回复模板**:
```
Hey! Welcome to Outbound! 🚐

I made a spoiler-free starter guide that covers exactly what to do in your first hour:
👉 https://outboundguide.org/en/starter-guide.html

Key tips:
- Don't waste rare materials early
- Prioritize power generation
- Check the crafting database before building

Hope it helps! Let me know if you have questions.
```

## 主动发帖策略

### 每周发布计划

**周一**: "本周热门问题汇总" - 整理本周讨论区最常见的问题并附带攻略链接

**周三**: "攻略更新通知" - 当有新补丁或新攻略发布时发帖

**周五**: "周末露营小贴士" - 分享一个具体的游戏技巧

---

### 发帖模板

**标题模板**:
- [Guide] Complete Crafting Database - Updated for 1.0.12
- [Tips] First Hour Walkthrough - What to Build First
- [Resource] Interactive Collectibles Map - All Hidden Items
- [Help] Multiplayer Troubleshooting Guide

**内容结构**:
1. 简短介绍（1-2句）
2. 攻略链接
3. 主要功能/内容列表
4. 邀请反馈

---

## 注意事项

1. **不要 spam** - 只在相关帖子下回复
2. **先提供价值** - 回答问题的核心，再附带链接
3. **保持友好** - 社区氛围很重要
4. **及时更新** - 当游戏更新时，及时更新攻略并通知社区
5. **尊重官方** - 明确说明是非官方攻略站
